2.2 智能推荐引擎 🔥 P0
2.2.1 引导式问答流程
*  第一步：产品类型选择
    *  展示主要产品分类
    *  图标化展示（易识别）
    *  支持搜索产品类型
    *  显示热门选择标识
    *  记录用户选择历史（快速再次选择）
*  第二步：自动信息填充
    *  自动读取用户宝宝信息
    *  显示当前使用的产品（基于历史评价）
    *  允许用户修改/确认信息
    *  显示为什么需要这些信息（提示文案）
*  第三步：关键问题引导
    *  使用场景选择（单选/多选）
    *  预算范围选择（滑块/区间）
    *  品牌偏好选择（可选）
    *  材质偏好选择（可选）
    *  特殊需求（开放输入）
    *  每个问题都有"跳过"选项
    *  问题数量控制在3-5个
*  进度显示
    *  显示当前第几步/共几步
    *  进度条可视化
    *  允许返回上一步修改
2.2.2 推荐算法逻辑
*  基础匹配规则
    *  月龄匹配（必须满足）
    *  价格匹配（在预算范围内）
    *  场景匹配（符合使用场景）
    *  过敏史匹配（排除不适合的）
*  权重计算
    *  同月龄宝妈推荐率（权重40%）
    *  产品综合评分（权重30%）
    *  品牌一致性加分（权重15%）
    *  评价数量加分（权重15%）
*  个性化调整 ⭐ P1
    *  基于用户历史浏览
    *  基于用户历史评价
    *  基于同类用户行为（协同过滤）
*  多样性保证
    *  不同价格段都有推荐
    *  不同品牌都有涵盖
    *  避免单一品牌霸屏
2.2.3 推荐结果展示 🔥 P0
*  筛选条件展示
    *  显示用户选择的所有条件
    *  一键修改筛选条件
    *  显示匹配产品数量
*  推荐理由说明
    *  为什么推荐这个产品（文案生成）
    *  有多少同月龄妈妈推荐
    *  突出核心优势（1-2个）
    *  使用场景匹配提示
*  排序方式
    *  默认：智能推荐排序
    *  按好评率排序
    *  按价格从低到高
    *  按价格从高到低
    *  按评价数量排序
    *  按销量排序 ⭐ P1
*  筛选器 ⭐ P1
    *  品牌筛选（多选）

* 1.1 双通道激励体系
* 
* 
* ┌─────────────────────────────────────┐
* │          宝妈参与平台                │
* │                                     │
* │   渠道A：内容贡献 → 积分奖励         │
* │   （评价、发帖、回答）               │
* │          ↓                          │

* │          ↓                          │
* │   现金提现或购物抵扣                 │

* 关键设计原则：
* ✅ 积分和佣金分离，避免套现漏洞
* ✅ 积分主要用于激励内容，佣金激励分享

* ## 三、佣金返现系统详细设计
* 
* ### 3.1 佣金分成规则
* 
* #### 【佣金来源】
* ```
* 淘宝联盟佣金（假设）：商品价格的10-15%
* 
* 平台获得：100%淘宝佣金
* ├─ 分给宝妈：80%
* ├─ 平台留存：20%
* └─ 第三方平台分成（如好单库）：已在淘宝佣金中扣除
* 
* 举例：
* 商品价格100元
* 淘宝佣金12元（12%）
* ├─ 宝妈得：9.6元（80%）
* ├─ 平台留：2.4元（20%）
* └─ 宝妈实际收益：9.6元
* ```
* 
* #### 【佣金计算逻辑】
* 
* **有效订单条件**
* ```
* ✅ 有效订单（计入佣金）：
* - 用户通过宝妈专属链接下单
* - 订单已付款
* - 订单确认收货（15天后）
* - 未发生退货退款
* - 在结算周期内
* 
* ❌ 无效订单（不计佣金）：
* - 用户未通过专属链接
* - 订单未付款
* - 订单退款/退货
* - 恶意刷单订单
* - 自己购买自己的链接
* ```
* 
* **佣金发放时机**
* ```
* 订单流程：
* 下单 → 付款 → 确认收货（+15天） → 佣金结算
* 
* 具体时间线：
* Day 0：用户下单付款
* Day 15：用户确认收货（或自动确认）
* Day 16-30：淘宝联盟结算期
* Day 31：佣金到账到宝妈账户（可提现）
* 
* 为什么30天后才能提现？
* - 防止退货（淘宝7天无理由退货期）
* - 等待淘宝联盟结算（通常需要20-30天）
* - 防止恶意套现
* 3.2 防刷机制（关键）
* 【自买自卖检测】
* 
* 
* javascript
* // 核心规则：严禁自己购买自己的推广链接
* 
* 检测逻辑：
* if (订单买家 === 推广者) {
*   订单标记为无效
*   不计入佣金
*   警告用户（3次封号）
* }
* 
* if (订单买家IP === 推广者IP) {
*   进入人工审核
*   要求提供关系证明
* }
* 
* if (订单买家设备 === 推广者设备) {
*   自动判定无效
* }
* 
* 允许的情况：
* - 宝妈推荐给家人/朋友（不同IP、不同设备）
* - 但需要留存聊天记录备查
* ```
* 
* #### 【刷单检测】
* ```
* 异常订单特征：
* - 同一买家短时间购买多个推广者的链接
* - 订单金额异常（过高或过低）
* - 收货地址异常（批量相同地址）
* - 买家账号新注册
* - 买家无真实消费记录
* 
* 处理机制：
* 1️⃣ 自动风控系统拦截
* 2️⃣ 人工审核订单真实性
* 3️⃣ 要求提供聊天记录/关系证明
* 4️⃣ 确认刷单：取消佣金+封号
* 5️⃣ 建立黑名单（买家+推广者）
* ```
* 
* #### 【恶意退货检测】
* ```
* 正常退货：
* - 产品质量问题
* - 尺寸不合适
* - 不喜欢
* → 合理，取消佣金，不处罚
* 
* 恶意退货：
* - 购买后立即退货（收货后1天内退货率>30%）
* - 多次购买同类产品并退货
* - 买家和推广者串通（同IP、频繁互动）
* → 取消佣金+警告+可能封号
* 
* 检测方法：
* - 追踪每个推广者的退货率
* - 退货率>20%触发审核
* - 退货率>40%暂停佣金发放
* ```
