宝妈说（平台级｜母婴真实决策平台）
核心定位： 基于同月龄真实使用反馈的母婴选品与避坑平台

一、平台一级入口（按「决策阶段」而不是按品类）
⚠️ 这是你和所有母婴平台最大的结构差异点 不是“卖什么”，而是“怎么做决定”

宝妈说
 ├─ Ⅰ. 怎么选（选前决策）
 ├─ Ⅱ. 怎么用（使用过程）
 ├─ Ⅲ. 值不值（复盘与评价）
 ├─ Ⅳ. 避坑榜（负反馈系统）
 └─ Ⅴ. 清单与工具（决策辅助）
下面我逐层展开。

二、Ⅰ. 怎么选（核心流量入口｜P0）
用户状态： “我要买，但我不知道怎么选、不想踩坑”

怎么选
 ├─ 1. 奶瓶怎么选（第一切口）
 │    ├─ 同月龄怎么选
 │    ├─ 品牌 / 型号对比
 │    ├─ 真实踩坑反馈
 │    ├─ 常见问题场景
 │    └─ 一句话结论
 │
 ├─ 2. 奶嘴怎么选
 │    ├─ 月龄 / 吸吮能力匹配
 │    ├─ 不同奶嘴差异
 │    ├─ 宝宝接受度反馈
 │    └─ 转换失败案例
 │
 ├─ 3. 奶瓶配件怎么选
 │    ├─ 适配关系表（核心）
 │    ├─ 场景解决方案
 │    ├─ 替代 / 升级建议
 │    └─ 风险与不适配说明
 │
 ├─ 4. 喂养相关用品怎么选
 │    ├─ 温奶器
 │    ├─ 消毒器
 │    ├─ 辅助喂养工具
 │    └─ 是否有必要买
 │
 └─ 5. 扩展品类（后续）
      ├─ 辅食工具怎么选
      ├─ 洗护用品怎么选
      └─ 出行用品怎么选
✅ P0 只做 1 + 3 其他是结构预留，不是同时做

三、Ⅱ. 怎么用（使用阶段信任强化｜P1）
用户状态： “我已经买了，但不知道用得对不对”

怎么用
 ├─ 1. 正确使用方法
 │    ├─ 官方说明拆解
 │    ├─ 宝妈真实用法
 │    └─ 易错点提醒
 │
 ├─ 2. 常见问题处理
 │    ├─ 胀气
 │    ├─ 拒奶
 │    ├─ 漏奶
 │    └─ 清洗麻烦
 │
 ├─ 3. 使用阶段建议
 │    ├─ 初次使用
 │    ├─ 稳定期
 │    └─ 转换期
 │
 └─ 4. 真实使用记录
      ├─ 使用天数
      ├─ 是否坚持使用
      └─ 是否中途放弃

四、Ⅲ. 值不值（真实评价系统｜平台根基）
用户状态： “这东西到底值不值？是不是被营销了？”

值不值
 ├─ 1. 真实评价
 │    ├─ 推荐 / 不推荐
 │    ├─ 使用时长
 │    ├─ 宝宝月龄
 │    └─ 一句话评价
 │
 ├─ 2. 深度反馈（进阶）
 │    ├─ 使用前预期
 │    ├─ 实际体验
 │    └─ 是否回购
 │
 ├─ 3. 对比评价
 │    ├─ 换过哪些
 │    ├─ 为什么换
 │    └─ 最终选择
 │
 └─ 4. 长期复盘
      ├─ 1 个月后
      ├─ 3 个月后
      └─ 是否推荐给同月龄

五、Ⅳ. 避坑榜（负反馈显性化｜差异点）
用户状态： “我只想知道哪些别买”

避坑榜
 ├─ 1. 同月龄避坑榜
 │    ├─ 0–3 个月
 │    ├─ 3–6 个月
 │    └─ 6–12 个月
 │
 ├─ 2. 场景避坑
 │    ├─ 胀气严重
 │    ├─ 奶嘴不接受
 │    └─ 夜奶失败
 │
 ├─ 3. 高频踩坑原因
 │    ├─ 设计问题
 │    ├─ 不适配
 │    └─ 夸大宣传
 │
 └─ 4. 被营销最多但不值的
⚠️ 这是 所有平台不敢做，但你必须做的模块
